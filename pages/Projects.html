Project: ATOL

An "Atol" is a group of islands. This is an engine and level editor for a platforming game, created using Processing. The idea was an atol being the setting for the game, with branching paths toward the shore of each island. Nicolas Moran and I authored the main program. Shoutout to Nick's brother, Alex Moran, for authoring a generic tile map editor for our adaptation. Project:ATOL can be found here.
